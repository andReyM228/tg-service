# Tg-Service

## Description
`tg-service` acts as the main service of the project, is responsible for the telegram bot.

## Technologies
- Go
- MongoDB
- Docker

## Architecture
Uses REST API to interact with `user-service` and `tx-service`.

## Launch
```bash
docker-compose up
```
